# Ikigai — AI-Powered Career Compass

Mobile-first Ikigai questionnaire that produces a **personalized career analysis** using [Inception Labs Mercury 2](https://www.inceptionlabs.ai/) — a diffusion LLM that's 5-10× faster than Claude or GPT.

The static archetype (Builder / Visionary / Changemaker / Strategist) is computed locally from the user's answers. The personalized analysis is generated by Mercury 2 based on the actual selections — not just the dominant circle.

---

## How it works

1. User answers 20 questions across 4 circles (Passion, Skills, Mission, Vocation).
2. Frontend computes engagement % per circle and shows the matching static archetype immediately.
3. Frontend POSTs the full answer payload to `/api/analyze`.
4. Cloudflare Pages Function (`functions/api/analyze.js`) builds a structured prompt from the answers and calls Mercury 2 via the Inception API.
5. The personalized analysis appears in the result screen as soon as it returns (typically 1-2 seconds — Mercury 2 is fast).

The Inception API key never reaches the browser — it's read from `env.INCEPTION_API_KEY` server-side in the Pages Function.

## Local development

You need [Node.js](https://nodejs.org/) installed.

### 1. Get an Inception API key

Sign up at [platform.inceptionlabs.ai](https://platform.inceptionlabs.ai/) — new accounts get **10M free tokens**, which is more than enough for personal use.

### 2. Set up local env

```bash
cp .dev.vars.example .dev.vars
```

Open `.dev.vars` and replace `your-inception-api-key-here` with your real key.

### 3. Run the dev server

```bash
npx wrangler pages dev . --port 8788
```

Wrangler serves the static files AND runs your Pages Function locally with the env vars from `.dev.vars`.

Open [http://localhost:8788](http://localhost:8788).

> If you don't want to install wrangler globally, `npx wrangler` will fetch it on first run.

### Alternative: static-only preview

If you just want to preview the UI without the AI analysis (the analysis section will show an error), you can use any static server:

```bash
npx serve .
# or
python3 -m http.server 8000
```

## Deploying to Cloudflare Pages

1. **Push to GitHub** — `.gitignore` already excludes `.dev.vars`, so your key won't be committed.

2. **Connect to Cloudflare Pages**:
   - Cloudflare dashboard → Workers & Pages → Create → Pages → Connect to Git
   - Pick your repo
   - **Framework preset**: None
   - **Build command**: leave empty (no build step needed)
   - **Build output directory**: `/` (root)

3. **Add the API key as a Secret** (most important step):
   - Project → Settings → Environment variables → Production
   - Variable name: `INCEPTION_API_KEY`
   - Value: your key from `platform.inceptionlabs.ai`
   - **Type: Secret** (not plain text)
   - Save and trigger a redeploy

4. **Test in production** — open your `*.pages.dev` URL, complete the questionnaire, the AI analysis should appear within a couple of seconds.

## Customizing

### Change the AI tone or analysis style

Edit `SYSTEM_PROMPT` at the top of `functions/api/analyze.js`. The current prompt asks for 4-5 paragraphs, no headers, with 2-3 specific career suggestions at the end.

### Switch model

In `functions/api/analyze.js`:

```js
const MODEL = 'mercury-2'  // change to 'mercury' for older but cheaper
```

Available Inception models: `mercury-2` (latest, $0.25/M input · $0.75/M output), `mercury` (older, slightly cheaper), `mercury-coder-small` (code-focused).

### Use a different LLM provider entirely

The Inception API is OpenAI-compatible, so swapping to OpenAI or any compatible provider is just changing `INCEPTION_URL`, the `Authorization` header format, and the model name. Anthropic Claude needs a different format (different header `x-api-key`, different request body shape).

### Add more questions

Edit the `QUESTIONS` array at the top of `app.js`. Each question has `circle`, `mode` (`"multi"` or `"single"`), `title`, and 5 `options`. The scoring logic adapts automatically.

## Cost & rate-limit notes

- **10M free tokens** on signup is enough for ~5,000 analyses (each ~2K tokens).
- After free tier, Mercury 2 is **$0.25 per 1M input tokens, $0.75 per 1M output tokens** — roughly $0.001 per analysis.
- The endpoint is public once deployed. If you share the URL widely, anyone can use it and bill your key. To restrict access, add Cloudflare Access (free for 50 users) or add a simple password check inside `functions/api/analyze.js`.

## Troubleshooting

**"Couldn't generate analysis"** — Open browser devtools → Network tab → click the `analyze` request → check the response. Most common: `INCEPTION_API_KEY` not set as a Secret in Cloudflare, or the key is invalid.

**Loading spinner stays forever** — The fetch failed silently. Check the browser console.

**Garbled characters in the original files** — The original files were Windows-1252 encoded; this version is converted to UTF-8.

**AI response is too short / too long** — Tune `max_tokens` (currently 1500) and the word count guidance in the system prompt.
